# Reactive Route

Documentation: https://dkazakov8.github.io/reactive-route/  
npm: https://www.npmjs.com/package/reactive-route

A reactive, framework-agnostic router in 2 KB.

**Reactive Route** is built for applications where routing should work naturally with reactive systems. 
It keeps route configuration explicit, strongly typed, and easy to maintain across different stacks.

Ready-made integrations are available for:

- React + MobX
- React + Observable
- Preact (no compat) + MobX
- Preact (no compat) + Observable
- Solid.js + Solid reactivity
- Solid.js + MobX
- Solid.js + Observable
- Vue + Vue reactivity

The library focuses on a small runtime, strong typing, mandatory URL parameter validation, 
fault tolerance, and support for SSR / Widget / MPA / no-JS.

No AI participated in the development.
